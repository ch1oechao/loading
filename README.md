# loading

### 快速增加一个 Loading 

    var loading = new Loading({
        container: $container, // loading 承载容器
        text: 'Loading...',    // loading 加载文字
        textColor: '#FFF',     // loading 文字颜色
        circleColor: '#FFF',   // loading 进度条颜色
        bgColor: 'pink',       // loading 背景颜色
        duration: '1s'         // loading 加载速度
    });

### API

- show() 显示 loading

- hide() 隐藏 loading
